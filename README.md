📈 Sales Prediction Using Machine Learning
📌 Project Overview
This project focuses on predicting product sales based on advertising expenditures across different marketing channels such as TV, Radio, and Newspaper. Using Machine Learning techniques, the model analyzes historical advertising data and predicts future sales with high accuracy.
The project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis, model training, evaluation, and prediction.
🎯 Objective
The primary goal of this project is to build a predictive model that helps businesses estimate sales based on advertising investments, enabling better marketing and business decisions.

Python

NumPy

Seaborn
Scikit-learn
Jupyter Notebook
📊 Project Workflow
1. Data Collection
Imported the sales dataset.
Examined dataset structure and features.
2. Data Preprocessing
Checked for missing values.
Cleaned and prepared data for analysis.
Selected relevant features.
3. Exploratory Data Analysis (EDA)
Visualized relationships between advertising channels and sales.
Analyzed trends and correlations.
Identified important factors affecting sales.
4. Model Building
Split data into training and testing sets.
Applied Linear Regression algorithm.
Trained the model using historical sales data.
5. Model Evaluation
Evaluated model performance using:
Mean Absolute Error (MAE)
Mean Squared Error (MSE)
R² Score
6. Sales Prediction
Predicted sales for new advertising budgets.
Compared predicted and actual values.
